# 🌐 The Meta Protocol
> **"Separate the Product from the Process."**

Le Meta Protocol est une méthodologie de développement assistée par IA (Vibe Coding) conçue pour maintenir des projets propres, scalables et sans pollution contextuelle.

## 🧠 Le Concept : La Sphère .meta/
Dans un projet standard, le chaos créatif (brouillons, notes de session, logs de debug) pollue le code source. Le Meta Protocol impose une séparation stricte :
- **src/ & docs/** : Le produit final (ce qui est livré).
- **.meta/** : La salle des machines (le cockpit du développeur et de l'IA).

## 🛠 Installation Rapide
1. Installer `copier` et `uv`.
2. `copier copy gh:ton-user/meta-protocol mon-nouveau-projet`
3. Répondez aux questions et commencez à piloter.
